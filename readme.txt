Automatically inserts `rel="nofollow"` into all the external links on your wordpress posts or pages.

== Description ==

<strong>If you need paid support to have this plugin installed or need to have a customized version please send an email to <a href="mailto:support@wordpressforge.com">support@wordpressforge.com</a></strong>.

Simple, if you use this plugin, `rel="nofollow"` will be inserted automatically for all the external links on your wordpress posts or pages.

= Feature =
* Adds `rel="nofollow"` to all external links in your wordpress posts/pages
* Will not add `rel="nofollow"` to any of the internal links of your wordpress posts/pages.
* Will not add another `rel="nofollow"` to links that already have `rel=nofollow`
* Will remove existing `dofollow` from `rel=""` if it exists within the link

You can visit our blog <a href="http://blog.wordpressforge.com/plugins/force-external-nofollow.html">blog.wordpressforge.com</a>

== Installation ==

1. Download the latest zip file and extract the 'wpf-force-external-nofollow' directory.
2. Upload it to your '/wp-content/plugins/' directory.
3. Activate 'WPF Force External Nofollow' in the 'Plugins' menu in WordPress.
